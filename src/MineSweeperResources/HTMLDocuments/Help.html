How to play MineSweeper

Clear all mines

There are mines hidden under some of these squares. Others have numbers. A number refers to the number of mines that are currently touching that square

Right click mines to mark them

Using right click on mines will put a flag on the square

Clear the board!

The game ends when only mines are left on the board

Extras

  • Press a number to clean the nearby zones. This will only work if there is the same number of flags nearby
  • Timer
  • Scoreboard
  • Beautiful themes
